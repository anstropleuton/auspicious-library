◆ word_wrap()

auto auspicious_library::sm::word_wrap ( std::string_view string,
std::size_t width,
bool force = false,
std::string_view delims = " \t\r\n\f\v\b" )
inlinenodiscardconstexpr

Word-wrap a string at width or before width depending on the delim.

Todo
Add to_string for std::string and char for consistency.

The string is tried to be split at the width if the character at the width is one of the delimiter. If not, the split occurs at the character before the width that is one of the delimiter. This process is repeated until the string is less than the size of width. If a word is longer than the specified width, the word is kept as is. If you use force, the word will be split.

Parameters
stringA string to word-wrap.
widthThe max word-wrap width.
forceWhether to force the string to always be less than or equal to the width (optional).
delimsThe delimiters, usually whitespace (optional).
Returns
result_string_nested of word-wrapped lines.
Note
word-wrapped lines can be larger than width.
