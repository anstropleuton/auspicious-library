Detailed Description

template<typename T, std::size_t Extent = std::dynamic_extent>
requires (std::is_default_constructible_v<T>)
struct auspicious_library::cu::boundless_span< T, Extent >

A boundless span.

Index-access of this span always returns a default constructed element when an invalid index is provided. Requires a default-constructible type.

Template Parameters
TThe type of element.
ExtentThe number of elements in the sequence, or std::dynamic_extent if dynamic (default).
