Detailed Description

template<typename CharT, typename Traits = std::char_traits<CharT>>
requires (std::is_default_constructible_v<CharT>)
struct auspicious_library::cu::boundless_basic_string_view< CharT, Traits >

A boundless basic string view.

Index-access of this string view always returns a default constructed element when an invalid index is provided. Requires a default-constructible type.
