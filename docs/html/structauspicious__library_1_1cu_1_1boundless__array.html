Detailed Description

template<typename T, std::size_t N>
requires (std::is_default_constructible_v<T>)
struct auspicious_library::cu::boundless_array< T, N >

A boundless array.

Index-access of this array always returns a default constructed element when an invalid index is provided. Requires a default-constructible type.

Template Parameters
TThe type of element.
NThe size of array.
