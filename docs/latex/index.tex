Auspicious Library is a collection of utils for Anstro Pleuton\textquotesingle{}s programs. This library contains several commonly used program artifacts in a simplified functions for ease of use.\hypertarget{index_autotoc_md54}{}\doxysection{\texorpdfstring{Sections}{Sections}}\label{index_autotoc_md54}
This library is subdivided into sections\+:
\begin{DoxyItemize}
\item {\bfseries{Container Utilities}} contains several utilities for container types (i.\+e., {\bfseries{std\+::vector}}, {\bfseries{std\+::array}}, etc. or custom compatible container types) which includes {\bfseries{appending elements}} (combining), {\bfseries{filtering elements out}}, etc. And several {\bfseries{operators}} for these operations.
\item {\bfseries{String Manipulators}} contains several utilities for {\bfseries{std\+::string}} (or {\bfseries{std\+::string\+\_\+view}} as parameters) which includes {\bfseries{converting containers to string}}, {\bfseries{word-\/wrap}}, {\bfseries{trimming string}}, converting {\bfseries{to lower case}}, etc. And several {\bfseries{operators}} from Container Utilities applied to string types.
\item {\bfseries{ANSI Escape Codes}} contains easy handlers for manipulation output using decorator \href{https://en.wikipedia.org/wiki/ANSI_escape_code}{\texttt{ ANSI Escape Codes}}.
\item {\bfseries{Argument Parser}} contains functionality to parse {\bfseries{Command Line Arguments}} and structures to {\bfseries{define options}} (or {\bfseries{switches}} if you are old and use Microsoft Windows) to easily validate arguments.
\item {\bfseries{File Utilities}} currently contains one function, to read all the file\textquotesingle{}s content in a string.
\end{DoxyItemize}\hypertarget{index_autotoc_md55}{}\doxysection{\texorpdfstring{Planned Sections}{Planned Sections}}\label{index_autotoc_md55}
There are several Utilities I am planning add to this library, notably\+:
\begin{DoxyItemize}
\item {\bfseries{Binary Files}} to contain several utilities for managing binary files to store and retrieve raw data.
\item {\bfseries{Security}} to contain encryption and decryption functions (yeah that\textquotesingle{}s it).

\begin{DoxyCode}{0}
\DoxyCodeLine{git\ clone\ https://github.com/anstropleuton/auspicious-\/library}
\DoxyCodeLine{cd\ auspicious-\/library}

\end{DoxyCode}

\end{DoxyItemize}
\item Build the library\+:
\begin{DoxyItemize}
\item Make build folder\+: 
\begin{DoxyCode}{0}

\DoxyCodeLine{cd\ build}

\end{DoxyCode}

\item Build the project\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{cmake\ ..}
\DoxyCodeLine{cmake\ -\/-\/build\ .}

\end{DoxyCode}

\end{DoxyItemize}
\item Install the library (optional, you can manually set up path if needed). {\bfseries{WARNING}}\+: Do not build tests or examples before installing.
