Detailed Description

template<typename CharT, typename Traits = std::char_traits<CharT>, typename Alloc = std::allocator<CharT>>
requires (std::is_default_constructible_v<CharT>)
struct auspicious_library::cu::boundless_basic_string< CharT, Traits, Alloc >

A boundless basic string.

Index-access of this string always returns a default constructed element when an invalid index is provided. Requires a default-constructible type.

Template Parameters
CharTThe character type.
TraitsThe character traits type.
AllocThe allocator type, defaults to std::allocator<CharT> .
